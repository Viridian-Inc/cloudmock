What This Means for the Desktop App

The desktop app is "Insomnia" to cloudmock's "Kong Gateway" — a rich, native tool for developers to interact with the platform. It's not just a dashboard viewer; it's where developers debug, inspect, and test their apps against cloudmock services.
cloudmock.io is "Konnect" — the hosted control plane where platform teams manage instances, set governance rules, onboard developers, and handle billing. It's a web console, not a debugging tool.
The gateway stays open-source — just like Kong Gateway. The monetization is in the hosted platform (cloudmock.io) and potentially premium desktop features.
