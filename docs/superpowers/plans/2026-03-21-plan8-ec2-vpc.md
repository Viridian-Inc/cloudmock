# EC2/VPC Tier 1 Promotion Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Promote EC2 from Tier 2 stub to full Tier 1 service with complete VPC networking (50+ actions, referential integrity, default VPC).

**Architecture:** Replace the EC2 stub entry with a hand-written Tier 1 service at `services/ec2/`. Uses query/form-encoded protocol with XML responses. All resources maintain cross-references and enforce dependency checks on deletion. Default VPC auto-created at service startup.

**Tech Stack:** Go 1.26, net/http, encoding/xml, crypto/rand, net (CIDR parsing), testify

**Spec:** `docs/superpowers/specs/2026-03-21-iac-tooling-design.md` — EC2/VPC section
