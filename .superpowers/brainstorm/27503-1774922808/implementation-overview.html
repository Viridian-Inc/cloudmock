Product Roadmap

Phase 1 — Desktop App MVP Current

Tauri shell, process manager, Activity + Services + Settings views, system tray

Phase 2 — Source SDKs + Full Observability

  • Node SDK (@cloudmock/node) — HTTP interception, console capture, errors
  • WebSocket source server in Rust, mDNS auto-discovery
  • Topology view (service graph + app nodes), Traces view, Metrics view

Phase 3 — Native SDKs + BLE + Remaining Views

  • Swift SDK, Kotlin SDK
  • BLE mesh scanning + topology visualization
  • SLOs, Incidents, Profiler, Chaos, AI Debug views
  • Proxy mode (zero-SDK network capture)

Phase 4 — cloudmock.io + Hosted Platform

Marketing site, web console, hosted instances, API key auth, team sharing, billing

Phase 5 — Developer Portal + Ecosystem

Service catalog, plugin marketplace, community plugins (GCP/Azure), third-party integrations
