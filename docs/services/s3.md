# S3 — Simple Storage Service

**Tier:** 1 (Full Emulation)
**Protocol:** REST-XML
**Service Name:** `s3`

## Supported Actions

| Action | Notes |
|--------|-------|
| `ListBuckets` | Returns all buckets in the account |
| `CreateBucket` | Creates a bucket; ignores LocationConstraint |
| `DeleteBucket` | Deletes an empty bucket |
| `HeadBucket` | Returns 200 if bucket exists, 404 otherwise |
| `PutObject` | Stores an object with arbitrary content |
| `GetObject` | Retrieves object content and metadata |
| `DeleteObject` | Removes a single object |
| `HeadObject` | Returns object metadata without body |
| `ListObjectsV2` | Lists objects with prefix/delimiter support |
| `CopyObject` | Server-side copy within or across buckets |

## Examples

### AWS CLI

```bash
# Create a bucket
aws s3 mb s3://my-bucket

# Upload a file
aws s3 cp ./data.json s3://my-bucket/data.json

# List objects
aws s3 ls s3://my-bucket/

# Download a file
aws s3 cp s3://my-bucket/data.json ./data-local.json

# Copy an object
aws s3 cp s3://my-bucket/data.json s3://my-bucket/backup/data.json

# Delete an object
aws s3 rm s3://my-bucket/data.json

# Delete a bucket and all contents
aws s3 rb s3://my-bucket --force
```

### Python (boto3)

```python
import boto3

s3 = boto3.client("s3", endpoint_url="http://localhost:4566",
                  aws_access_key_id="test", aws_secret_access_key="test",
                  region_name="us-east-1")

# Create bucket
s3.create_bucket(Bucket="my-bucket")

# Put object
s3.put_object(Bucket="my-bucket", Key="hello.txt", Body=b"Hello, world!")

# Get object
response = s3.get_object(Bucket="my-bucket", Key="hello.txt")
print(response["Body"].read())  # b"Hello, world!"

# List objects
result = s3.list_objects_v2(Bucket="my-bucket")
for obj in result.get("Contents", []):
    print(obj["Key"])
```

## Notes

- Bucket names must be unique within the emulated account.
- Object metadata (Content-Type, ETag, LastModified) is stored and returned.
- `ListObjectsV2` supports `Prefix` and `Delimiter` query parameters.
- Versioning and bucket policies are not implemented.
